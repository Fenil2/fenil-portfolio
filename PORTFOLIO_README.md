# 🚀 Professional Web Developer Portfolio

A stunning, modern portfolio website built with **Next.js 15**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## ✨ Features

### 🎨 Design
- **Exact replica** of professional Webflow template
- **Geist font** (Google Fonts) - modern, clean typography
- **Dark gradient theme** with precise color matching
- **Fully responsive** - mobile, tablet, desktop optimized
- **Crystal-clear images** with Next.js optimization

### ⚡ Animations (Framer Motion)
- **Professional page load sequence** with staggered elements
- **Parallax scrolling** for depth and immersion
- **Floating cartoon animation** (4s subtle loop)
- **Infinite scroll button bounce** with smooth easing
- **Interactive hover states** on all clickable elements
- **Smooth transitions** with custom cubic-bezier timing
- **Pulsing glow effects** beneath hero image
- **GPU-accelerated** for 60fps performance

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Open browser
http://localhost:3000
```

## 🎬 Animation Highlights

### Page Load (Choreographed)
1. Header slides down (600ms)
2. White line expands (1000ms)
3. Heading fades in with stagger
4. Cartoon scales + fades (1000ms)
5. Scroll button appears (800ms)
6. Badge final reveal (1200ms delay)

### Continuous Motion
- **Cartoon float**: 4s gentle up/down
- **Glow pulse**: 3s opacity/scale cycle
- **Arrow bounce**: 1.5s infinite loop

### Interactive
- Logo spins 360° on hover
- Links show animated underline
- Arrows slide right + gap expands
- All buttons have tap feedback
